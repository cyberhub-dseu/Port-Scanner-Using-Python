# Port Scanner

A simple port scanner written in Python that scans for open ports on a specified IP address or domain.

## Features

- Scans ports from 1 to 1024
- Displays open ports
- Error handling for hostname resolution

## Requirements

- Python 3.x
- `pyfiglet` library
